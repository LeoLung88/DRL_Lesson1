台北時間自然時鐘 (Taipei Time Natural Clock)

深度強化學習專案 - 網頁示範 (Deep Reinforcement Learning Project - Web Demo)

作品介紹 (Introduction)

這是一個以乾淨、現代化設計風格打造的單頁面「台北時間自然時鐘」。專為提供最佳的視覺體驗而設計，結合了動態特效與深色模式的美學。

主要特色：

  • 精準的台北時間： 使用 JavaScript Intl.DateTimeFormat API 鎖定 Asia/Taipei 時區，不受使用者本地端時區影響，隨時呈現精準的台灣時間。
  • 玻璃擬物化設計 (Glassmorphism)： 時鐘面板採用高質感的半透明模糊背景，搭配柔和的漸層與網格光效。
  • 動態視覺回饋： 滑鼠懸停時會觸發流暢的立體浮動動畫 (Elevate on hover)，並伴隨背景光暈的呼吸燈脈衝效果。
  • 極簡佈局： 將時鐘、日期完美置中於大正方形內，並上方特別標註 TAIPEI TIME。
