# 台北時間自然時鐘 (Taipei Time Natural Clock)

一個使用 HTML, CSS 與 JavaScript 打造的現代化網頁時鐘，專為展示台北時間而設計。此專案為深度強化學習課程 (DRL Lesson 1) 之網頁示範。

## 🌟 作品特色

- **純手工打造 (No Dependencies)**：不依賴任何外部框架 (如 React / Vue 或 Tailwind)，完全使用原生 HTML, CSS 與 Vanilla JS 撰寫。
- **高質感視覺設計 (Glassmorphism)**：結合半透明毛玻璃效果、深色模式與環境光暈 (Glow effect)，展現極致的現代化視覺美學。
- **流暢且細緻的動畫回饋**：滑鼠懸浮時的 3D 立體浮雕動畫 (Elevate on hover)、背景呼吸燈脈衝及秒針冒號閃爍效果。
- **精準時區鎖定**：利用 JavaScript 原生 `Intl.DateTimeFormat` API 完美鎖定 `Asia/Taipei`，確保不受觀看者本地設備時區影響，隨時呈現精準的台灣時間。
- **個人化專屬標誌**：將時鐘標題客製化為專屬的「LUNG HAO WEN」動態字體設計。

## 📂 檔案架構

- [`taipei_clock.html`](taipei_clock.html)：專案核心檔案，包含所有的 HTML 結構、CSS 樣式與時鐘運作邏輯。
- [`index.html`](index.html)：展示核心專案的 Demo 說明網頁，詳細介紹專案特色，並透過 `<iframe>` 完美嵌入時鐘本體進行展示。
- `README.md`：專案說明文件（本專案導覽）。

## 🚀 如何啟動

不需要任何編譯或環境配置！
